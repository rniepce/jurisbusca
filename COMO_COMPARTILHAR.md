# Assistente TJMG 2.0 - Protótipo

Este é o protótipo de alta fidelidade do novo Assistente Judicial Inteligente.

## Como Visualizar

1. **Localmente (Seu Computador)**:
   - Abra a pasta do projeto.
   - Dê um duplo clique no arquivo `index.html`.
   - O protótipo abrirá no seu navegador padrão (Chrome, Edge, Safari, etc).

## Como Compartilhar com Outros

### Opção 1: Enviar Arquivo (Simples)
1. Comprima (Zip) esta pasta inteira (contendo `index.html`, `css` e `js`).
2. Envie o arquivo `.zip` por E-mail, WhatsApp ou Drive.
3. A pessoa deverá descompactar e abrir o `index.html`.

### Opção 2: Link na Web (Mais Profissional)
Para gerar um link que qualquer pessoa pode acessar sem baixar nada:

1. Acesse o site **[Netlify Drop](https://app.netlify.com/drop)**.
2. Arraste a pasta `assistente` para a área indicada.
3. Em segundos, ele gera um link (ex: `site-legal.netlify.app`).
4. Copie esse link e envie para os colegas.
